Project ELT Pipeline with Airflow, PostgreSQL, and dbt

This project demonstrates an end-to-end ELT (Extract, Load, Transform) data pipeline built using Docker, Apache Airflow, PostgreSQL, and dbt. The pipeline automates the ingestion of raw transactional data from a source database, loads it into a data warehouse, and applies transformations — including data denormalization — to produce analytics-ready tables.
